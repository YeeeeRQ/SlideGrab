# SlideGrab - 视频PPT提取器

让视频中的PPT瞬间成为可编辑的演示文稿

【为什么选择SlideGrab】
- 从此告别手动截图 - 自动识别视频中的每个幻灯片
- 3种智能算法 - 适配不同类型的视频内容
- 一键导出PPT - 提取的幻灯片直接生成可编辑的演示文稿

【主要功能】
- 智能场景检测：直方图法、帧差分法、差异哈希法
- 手动精确提取：支持逐帧选择关键内容
- 批量导出：支持PNG、JPG图片格式和PPT演示文稿
- 幻灯片管理：网格/列表视图，预览、选择、删除

【适用人群】
- 在线教育工作者
- 企业培训师
- 知识博主
- 学术研究人员
- 会议记录专员

## 技术栈

- React + TypeScript + Vite
- Tailwind CSS
- shadcn/ui

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
