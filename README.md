# SlideGrab - 视频PPT提取器

从视频中提取PPT幻灯片，导出为图片或PPT。

## 功能特性

- 手动提取帧
- 自动检测（直方图法、帧差分法、差异哈希法）
- 幻灯片管理（网格/列表视图）
- 预览功能（缩放、拖拽）
- 导出支持（PNG、JPG、PPT）
- 响应式布局

## 技术栈

- React + TypeScript + Vite
- Tailwind CSS
- shadcn/ui

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
